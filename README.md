## Description

Technical test PT. EIGEN TRI MATHEMA.

## Installation

Clone project

```bash
$ git clone https://github.com/raizafr/technical-test-fe-eigen.git
$ cd technical-test-fe-eigen.git
```

Install semua dependencies

```bash
$ npm install
```

## Add .env file di root project

Create file `.env`

```bash
$ API_BASEURL = https://newsapi.org/v2/
$ API_KEY = your api key
```

## Running the app

```bash
# development
$ npm run dev

# build app
$ npm run build

# production mode
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# watch
$ npm run test:watch

# coverage
$ npm run test:cover
```
